# Robot Path Planning Visualizer

A simple web-based robot navigation visualizer for exploring shortest-path planning using BFS and A* search.

## Features

- Interactive grid with start, goal, and obstacle placement.
- BFS and A* algorithms for path planning.
- Visual feedback for open set, closed set, and final path.

## Usage

1. Open `index.html` in a browser.
2. Click `Set Start` and choose a starting cell.
3. Click `Set Goal` and choose a destination cell.
4. Use `Add/Remove Obstacles` to place walls.
5. Click `Run BFS` or `Run A*` to find a path.
6. Use `Clear Path` to reset the visualization, or `Reset Grid` to start over.
